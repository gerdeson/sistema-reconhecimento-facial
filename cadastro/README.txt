PASTA DE CADASTRO
================

Esta pasta deve conter as fotos das pessoas que você quer reconhecer.

INSTRUÇÕES:
1. Adicione uma foto por pessoa
2. Nomeie os arquivos com o nome da pessoa (ex: joao_silva.jpg, maria_santos.png)
3. Use fotos com o rosto bem visível e boa iluminação
4. Formatos suportados: .jpg, .jpeg, .png, .bmp, .tiff, .tif

EXEMPLOS DE NOMES DE ARQUIVO:
- joao_silva.jpg
- maria_santos.png
- carlos_oliveira.jpg
- ana_pereira.png

DICAS PARA MELHOR RECONHECIMENTO:
- Use fotos de boa qualidade
- Evite fotos muito escuras ou com muita sombra
- O rosto deve estar bem visível e centralizado
- Evite fotos com óculos escuros ou acessórios que cubram o rosto

Após adicionar as fotos, execute:
python cadastro_simples.py --mode setup

Ou use o script executar.bat e escolha a opção 2.
